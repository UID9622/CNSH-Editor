## 📝 常见问题

### Q1: 双击没反应？

**A:** 检查浏览器
- Windows: 右键 → 打开方式 → Chrome/Firefox
- Mac: 右键 → 打开方式 → Safari/Chrome

### Q2: 编译失败？

**A:** 检查语法
- 使用中文方括号【】
- 打印用「」包裹字符串
- 函数后加"返回类型 整数"

### Q3: 怎么下载C代码？

**A:** 
1. 编译成功后
2. 切换到"⚙️ C代码"标签
3. 复制粘贴到文本文件
4. 保存为.c文件

### Q4: 可以离线使用吗？

**A:** 
- v1.0: ✅ 完全离线
- v2.0: ⚠️ 需要网络（首次加载CodeMirror）
