## 🔥 这是什么？

**实话实说**：这不是一个完整的代码编辑器（那是未来的事），  
这是一个**数字甲骨文字元创作工具**——让你用鼠标就能创作属于自己的汉字字元。

### 你能用它做什么？

- ✅ **鼠标绘制字元**：在画布上拖动鼠标，画出笔画
- ✅ **15层渲染特性**：力度、侵蚀、纹理、墨色...一键调节
- ✅ **导出 SVG 矢量图**：可打印、可分享、可商用
- ✅ **保存 .cnsh 格式**：可重新编辑
- ✅ **DNA 追溯**：每个字元都有唯一身份标识

### 你不能用它做什么？

- ❌ 写代码（Python/JavaScript 那种）
- ❌ 编程逻辑（if/else/for 循环）
- ❌ 批量生成字体包

**这些功能在路线图上，会一步步实现。现在发布的是第一步：让你能创作字元。**

## 🚀 怎么用？

### 1. 下载并解压

解压后你会看到：
```
CNSH_军人编辑器_普惠版_v0.1/
├── 编辑器/
│   ├── index.html         # 主编辑器
│   ├── 快速启动.html      # 引导页面
│   └── 使用说明.md        # 详细文档
├── 示例作品/               # 示例 SVG
└── README.md              # 本文件
```

### 2. 打开编辑器

**双击 `编辑器/快速启动.html`**，浏览器会自动打开。

### 3. 开始创作

1. 在"字元名称"输入框填写你要创作的字（如：爱、和、平）
2. 在画布上按住鼠标左键拖动，绘制笔画
3. 调整参数：
   - 笔画力度（8-30）
   - 侵蚀强度（0-1.0，风化效果）
   - 纹理类型（光滑/粗糙/石刻）
   - 墨色浓度（0.4-1.0）
4. 保存作品（.cnsh JSON）或导出 SVG

### 4. 分享你的作品

- 打印出来
- 分享给朋友
- 用在设计项目中
- 上传到社区（未来会有）

每个字元都经过三色审计：
- 🔴 **红线审计**：禁止西方术语、英文变量
